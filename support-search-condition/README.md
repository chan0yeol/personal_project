# UniSupport 검색 조건 Extension

유니서포트 사이트(`114.unipost.co.kr`)에서 검색 조건을 빠르게 설정하고 조회할 수 있는 Chrome 익스텐션입니다.

---

## 설치 방법

1. Chrome 브라우저에서 주소창에 `chrome://extensions` 입력
2. 우측 상단 **개발자 모드** 토글 ON
3. **압축해제된 확장 프로그램 로드** 클릭
4. 아래 경로 입력 후 선택

```
\\dockersys.synology.me\maintenance\team6\common\support-search-condition
```

> 네트워크 드라이브가 연결되어 있어야 합니다.

---

## 버튼 설명

사이트 접속 시 화면 상단에 버튼 3개가 나타납니다.

| 버튼 | 설명 |
|------|------|
| **날짜 조회** | 저장된 날짜 범위 + 팀 조건으로 조회 (처리자 필터 없음 · 전체 라디오) |
| **설정한 조건 조회** | 저장된 날짜 범위 + 팀 + 처리자 조건으로 조회 |
| **검색 설정** | 조회 조건 설정 모달 열기 |

---

## 검색 설정 모달

**검색 설정** 버튼을 누르면 아래 항목을 설정할 수 있습니다.

### 처리자
- 텍스트 직접 입력
- 저장된 값이 없으면 현재 로그인 계정 이름이 자동으로 입력됩니다

### 시작일
조회 시작일을 오늘 기준으로 선택합니다.

| 옵션 | 기간 |
|------|------|
| 10일 전 | 오늘 기준 10일 전 (기본값) |
| 한달 | 30일 전 |
| 3개월 | 90일 전 |
| 6개월 | 180일 전 |
| 360일 | 360일 전 |

> 종료일은 항상 **오늘 날짜**로 자동 설정됩니다.

### 팀
조회할 팀을 복수 선택할 수 있습니다.

- **전체** 클릭 → 1~7팀 + 미지정 전체 선택
- 개별 팀 선택 시 해당 팀만 조회

### 화면 진입 시 자동 설정
- 토글 **ON** : 페이지 접속할 때마다 저장된 조건으로 자동 조회 실행
- 토글 **OFF** : 버튼 클릭 시에만 조회

### 버튼
| 버튼 | 동작 |
|------|------|
| 닫기 | 저장 없이 모달 닫기 |
| 저장 | 설정만 저장 (조회 실행 안 함) |
| 저장 & 조회 | 설정 저장 후 즉시 조회 실행 |

---

## 설정 저장 방식

설정값은 브라우저 **쿠키**에 저장됩니다. 브라우저를 닫아도 유지되며, 다음 접속 시 자동으로 불러옵니다.
