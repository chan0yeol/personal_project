## 0. 이 문서를 읽는 Claude에게

이 문서는 다음 세션에서 바로 개발을 이어갈 수 있도록 모든 결정사항과 맥락을 담고 있습니다.

**핵심 결정사항 요약:**
- 모든 코드는 **Docker Compose 기반**
- AI API 비용 없음 → **Claude Code 스케줄 에이전트** ($20 Pro, 5회/일) 로 스크립트 자동 생성
- 생성된 스크립트는 **GitHub 레포에 JSON으로 저장**
- 로컬 Docker가 GitHub에서 pull → 영상 제작 → 유튜브 업로드 자동화
- 블로그 자동화도 동일 구조로 확장 예정 (2단계)
- 다음 할 일: **GitHub 레포 생성 → Docker 구조 세팅 → 첫 채널(역사 속 오늘) 개발**

## 1. 프로젝트 개요

### 핵심 컨셉

**완전 자동화** - 수동 작업 0, AI API 비용 0

```
[Anthropic 클라우드]               [로컬 - Docker Compose]
스케줄 에이전트                         scheduler
(하루 최대 5회 실행)   → GitHub →       generator
채널별 스크립트 자동 생성   JSON          uploader
+ GitHub 자동 커밋                   (매일 새벽 자동 실행)
```

## 2. 전체 시스템 구조

### 파이프라인 흐름

```
① 스케줄 에이전트 (Anthropic 클라우드, 매일 자동 실행)
   └─ 채널별 스크립트 7개 생성
   └─ scripts/{채널명}/YYYY-WW.json 으로 GitHub 커밋

② Docker scheduler (로컬, 매일 새벽 실행)
   └─ GitHub에서 git pull (최신 스크립트 동기화)
   └─ 오늘 날짜에 해당하는 스크립트 선택

③ Docker generator
   └─ 스크립트 JSON 읽어서 영상 제작
      - TTS 음성 생성 (gTTS, 무료)
      - 자막 + 배경 합성 (MoviePy + Pillow)
      - 썸네일 생성 (Pillow)
   └─ /volumes/output/ 에 영상 저장

④ Docker uploader
   └─ YouTube Data API v3로 영상 업로드
   └─ 제목 / 설명 / 태그 / 썸네일 자동 설정
```

### 스케줄 에이전트 할당 (5회/일)

| 실행 횟수 | 채널 | 생성량/일 | 월 생성량 |
|----------|------|----------|---------|
| 1회 | 역사 속 오늘 | 7개 | 210개 |
| 2회 | 영어 표현 | 7개 | 210개 |
| 3회 | 명언 / 동기부여 | 7개 | 210개 |
| 4회 | 뉴스 요약 | 7개 | 210개 |
| 5회 | 예비 / 신규 채널 | 7개 | 210개 |

## 3. 채널 주제 전략

### A티어 - MVP 시작 채널

#### 3-1. 역사 속 오늘 (첫 번째 채널)
- **포맷**: "XX년 오늘, [사건]이 일어났습니다"
- **영상 구성**: 배경이미지 + 자막 + TTS 나레이션 (60초)
- **업로드 주기**: 매일 1개
- **자동화 난이도**: ★☆☆
- **특징**: 365일치 콘텐츠 무한 반복 가능, 저작권 이슈 없음

#### 3-2. 오늘의 영어 표현
- **포맷**: "원어민이 매일 쓰는 영어 표현 1개"
- **영상 구성**: 텍스트 애니메이션 + 예문 + TTS (영어/한국어)
- **업로드 주기**: 매일 1개
- **자동화 난이도**: ★☆☆
- **특징**: 구독자 락인 효과, 꾸준한 수요

#### 3-3. 오늘의 명언 (동기부여)
- **포맷**: 명언 + 배경음악 + 감성 배경
- **영상 구성**: 텍스트 페이드인 + 배경음악
- **업로드 주기**: 매일 1개
- **자동화 난이도**: ★☆☆

#### 3-4. 오늘 뉴스 TOP5 요약
- **포맷**: "오늘 꼭 알아야 할 뉴스 5가지"
- **영상 구성**: 헤드라인 카드 슬라이드 + 자막
- **업로드 주기**: 매일 1~2개
- **자동화 난이도**: ★★☆

### B티어 - 2단계 확장

#### 3-5. 세계 랭킹 / 통계 시각화
- **포맷**: "세계 [주제] 순위 TOP10"
- **영상 구성**: Bar Chart Race 애니메이션
- **자동화 난이도**: ★★★

#### 3-6. 주식 / 코인 시황 요약
- **포맷**: "오늘 코스피 / 비트코인 요약"

## 4. 기술 스택

### Docker Compose 서비스 구성

```yaml
services:
  scheduler:    # 매일 새벽 git pull + 파이프라인 트리거
  generator:    # 영상 제작 (MoviePy + gTTS + Pillow)
  uploader:     # 유튜브 업로드 (YouTube Data API v3)
  redis:        # 서비스 간 메시지 큐
```

### 사용 라이브러리

```
[영상 제작 - 비용 0원]
- MoviePy        # 영상 편집/합성
- Pillow / PIL   # 이미지/썸네일 생성
- gTTS           # TTS 음성 생성 (무료)
- FFmpeg         # 인코딩 (컨테이너 내 설치)

[업로드]
- google-api-python-client   # YouTube Data API v3

[스케줄링]
- APScheduler    # 매일 새벽 자동 실행

[유틸]
- Redis          # 서비스 간 메시지 큐
- GitPython      # GitHub 자동 pull
```

### 스크립트 JSON 포맷

```json
{
  "channel": "history-today",
  "generated_at": "2026-04-16",
  "scripts": [
    {
      "id": "20260416",
      "date": "04-16",
      "title": "1912년 오늘, 타이타닉이 침몰했습니다",
      "narration": "1912년 4월 15일 새벽 2시 20분, 역사상 가장 큰 여객선이었던 타이타닉호가 북대서양 차가운 바다 속으로 사라졌습니다...",
      "subtitle_lines": [
        "1912년 4월 15일",
        "타이타닉 침몰",
        "1,500명 사망"

      "tags": ["역사", "오늘의역사", "타이타닉", "역사속오늘"],
```

## 5. 스케줄 에이전트 설정

### 에이전트 개요
- **플랫폼**: Claude Code 스케줄 원격 에이전트
- **실행 위치**: Anthropic 클라우드 (내 PC 꺼져도 동작)
- **할당량**: 5회/일 (Claude Pro $20 구독)
- **저장소**: GitHub 레포 (무료)

### 에이전트 프롬프트 템플릿 (역사 속 오늘)

```
이번 주 "역사 속 오늘" 유튜브 쇼츠 스크립트 7개를 생성하고
scripts/history-today/YYYY-WW.json 파일로 저장 후 커밋해줘.

조건:
- 각 날짜별 임팩트 있는 역사적 사건 1개
- narration은 60초 분량 (약 150자)
- 쇼츠에 맞게 흥미롭고 간결하게
- title, narration, subtitle_lines(3줄), tags, thumbnail_text, bg_keyword 포함

출력: 위 JSON 포맷으로 파일 저장 후 GitHub 커밋
```

## 7. 확장 계획 (2단계)

### 블로그 자동화 연동
동일한 스케줄 에이전트 구조로 블로그도 자동화 가능

```
스케줄 에이전트
├── 쇼츠 스크립트 생성 → GitHub → Docker → 유튜브
└── 블로그 글 생성    → GitHub Pages / Tistory API → 자동 발행

수익: 유튜브 광고 + 블로그 애드센스 + SEO 유입
```
