# 🪟 PiP Overlay

브라우저 위에 띄우는 **반투명 플로팅 미니 브라우저** 크롬 익스텐션.

영상, 문서, 웹사이트 등 어떤 URL이든 현재 페이지 위에 오버레이로 띄우고,  
투명도·위치·크기를 자유롭게 조절할 수 있습니다.

---

## 주요 기능

- **플로팅 미니 브라우저** — 현재 탭 위에 iframe 오버레이로 렌더링
- **투명도 조절** — 하단 슬라이더로 10~100% 실시간 조절
- **드래그 이동** — 상단 툴바를 드래그해서 위치 변경
- **리사이즈** — 우하단 핸들로 자유롭게 크기 조절
- **X-Frame-Options 우회** — YouTube 등 iframe 차단 사이트도 로드 가능
- **숨기기 / 보이기** — 팝업 버튼으로 토글

---

## 설치

1. 크롬 주소창에 `chrome://extensions` 입력
2. 우상단 **개발자 모드** 활성화
3. **압축해제된 확장 프로그램 로드** 클릭
4. 이 폴더(`pip-extension`) 선택

---

## 사용법

### 오버레이 열기
1. 크롬 툴바에서 🪟 아이콘 클릭
2. URL 입력 (선택) → **오버레이 열기** 클릭
3. URL 없이 열면 빈 브라우저로 시작

### 오버레이 조작

| 동작 | 방법 |
|---|---|
| 이동 | 상단 툴바 드래그 |
| 리사이즈 | 우하단 `⊿` 드래그 |
| 투명도 조절 | 하단 슬라이더 (10~100%) |
| URL 이동 | URL 입력창에 입력 후 Enter |
| 뒤로가기 | `‹` 버튼 |
| 새로고침 | `↺` 버튼 |
| 숨기기/보이기 | 팝업 → 숨기기/보이기 버튼 |
| 닫기 | 툴바 `✕` 버튼 |

## 파일 구조

```
pip-extension/
├── manifest.json       # 익스텐션 설정 및 권한 선언
├── background.js       # X-Frame-Options 응답 헤더 제거 (iframe 차단 우회)
├── content.js          # 플로팅 오버레이 UI 전체 로직
├── popup.html          # 툴바 팝업 UI
└── popup.js            # 팝업 이벤트 핸들러
```

---

## 기술 메모

- **Manifest V3** 기반
- `declarativeNetRequest` 로 `X-Frame-Options` / `Content-Security-Policy` 응답 헤더를 `sub_frame` 요청에서 제거 → 대부분의 사이트 iframe 로드 가능
- 드래그/리사이즈 중 iframe이 마우스 이벤트를 가로채는 문제는 투명 blocker div로 해결
- `chrome.runtime.onMessage` 로 팝업 → content script 간 URL 전달

---

## 알려진 한계

- 오버레이는 **현재 탭 안에서만** 유지됨 (탭 전환 시 사라짐)
- 일부 사이트는 JS로 iframe 탈출을 시도할 수 있음
- 크롬 내부 페이지(`chrome://`)에서는 content script 동작 안 함
